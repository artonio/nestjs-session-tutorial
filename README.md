# session-tutorial

## Description

using basic auth to authenticate and use express-session to manage sessions with http cookie

Source code for NestJS Basic Auth and Sessions Tutorial http://blog.exceptionfound.com/index.php/2018/06/07/nestjs-basic-auth-and-sessions/

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
